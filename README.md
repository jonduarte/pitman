# Pitman

A simple command line tool to help users find unused gems.

## Installation

Add this line to your application's Gemfile:

```ruby
gem install pitman
```

And then execute:

    $ pitman <DIR>


## Usage

<DIR> should be the main directory where you place all your ruby projects.
